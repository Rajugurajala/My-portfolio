    project 1:Simple Interest Calculator

    (link:-https://rajugurajala.github.io/project1/)

    Description About Project

    This project is a user-friendly "Simple Interest Calculator" designed to compute interest and total amount based on user inputs. It features input fields for the initial amount, interest rate, total months, and total days, providing flexibility for various interest calculation scenarios. The calculator dynamically updates the months field if the number of days exceeds 30, automatically converting excess days into months for accurate calculations. Error handling is implemented to alert users if the initial amount or interest rate fields are left empty, ensuring valid inputs for calculations. The user interface is responsive, adapting its layout and element sizes for optimal viewing across different devices, from small smartphones to larger screens. Navigation is enhanced with keyboard accessibility, allowing users to efficiently move between input fields using the "Enter" key. The project provides clear output, displaying both the calculated total interest and the final amount including interest, making the results easily understandable.

    Technologies Used: HTML for structure, CSS for styling and responsiveness, and JavaScript for the calculation logic and interactive elements.
